## 第 1 题答案 — 编程题 — ★★☆

**参考实现：**

```c
#define BLOCK_SIZE  64
#define BLOCK_COUNT 16

// 确保 block 对齐要求
#define BLOCK_ALIGN  __attribute__((aligned(8)))

typedef struct MemBlock {
    struct MemBlock *next;  // 空闲链表的 next 指针
    // ... 后面是用户数据
} MemBlock;

typedef struct {
    MemBlock *free_list;
    uint8_t   pool_mem[BLOCK_COUNT][BLOCK_SIZE] BLOCK_ALIGN;
    uint32_t  total_blocks;
    uint32_t  free_blocks;
} MemPool;

void mempool_init(MemPool *pool) {
    pool->free_list = (MemBlock *)pool->pool_mem;
    pool->total_blocks = BLOCK_COUNT;
    pool->free_blocks = BLOCK_COUNT;
    
    // 串联所有空闲块
    uint8_t *block_ptr = pool->pool_mem;
    for (int i = 0; i < BLOCK_COUNT - 1; i++) {
        MemBlock *block = (MemBlock *)block_ptr;
        block->next = (MemBlock *)(block_ptr + BLOCK_SIZE);
        block_ptr += BLOCK_SIZE;
    }
    // 最后一块的 next = NULL
    ((MemBlock *)(pool->pool_mem + (BLOCK_COUNT - 1) * BLOCK_SIZE))->next = NULL;
}

void *mempool_alloc(MemPool *pool) {
    if (!pool->free_list) return NULL;  // 池已空
    
    MemBlock *block = pool->free_list;
    pool->free_list = block->next;
    pool->free_blocks--;
    
    return (void *)block;
}

void mempool_free(MemPool *pool, void *ptr) {
    if (!ptr) return;
    
    MemBlock *block = (MemBlock *)ptr;
    block->next = pool->free_list;
    pool->free_list = block;
    pool->free_blocks++;
}

// 诊断函数
uint32_t mempool_free_count(MemPool *pool) {
    return pool->free_blocks;
}

bool mempool_contains(MemPool *pool, void *ptr) {
    uintptr_t start = (uintptr_t)pool->pool_mem;
    uintptr_t end = start + BLOCK_COUNT * BLOCK_SIZE;
    uintptr_t p = (uintptr_t)ptr;
    return (p >= start && p < end);
}
```

**详细解析：**

固定大小内存池是嵌入式系统的核心数据结构之一：
1. **O(1)** 分配/释放，无碎片
2. **确定性强**：分配永远不会失败（除非池已空）
3. **无需 malloc**：所有内存在 .bss 段
4. **next 指针复用**：空闲时用 block 开头存储 next 指针；分配后给用户用，不浪费空间

适用于消息队列缓冲区、网络包缓冲区、对象池等场景。

## 第 2 题答案 — 编程题 — ★☆☆

**参考实现：**

```c
#define DEBOUNCE_MS 20

typedef struct {
    uint8_t  stable_state;   // 去抖后稳定状态 (0/1)
    uint8_t  last_stable;    // 上一次稳定状态（用于边沿检测）
    uint8_t  debounce_cnt;   // 消抖计数器
    uint8_t  raw_last;       // 上一次原始电平
} ButtonDebounce;

// 每 1ms 调用
bool button_debounce(ButtonDebounce *btn, bool raw) {
    // 电平变化 → 重置计数器
    if (raw != btn->raw_last) {
        btn->debounce_cnt = 0;
        btn->raw_last = raw;
    }
    // 电平持续稳定 → 计数器递增
    else {
        if (btn->debounce_cnt < DEBOUNCE_MS) {
            btn->debounce_cnt++;
        }
        // 达到消抖时间 → 确认状态
        if (btn->debounce_cnt >= DEBOUNCE_MS) {
            btn->last_stable = btn->stable_state;
            btn->stable_state = raw;
        }
    }
    return btn->stable_state;
}

// 检测按下事件（高→低）
bool button_is_pressed(ButtonDebounce *btn) {
    return (btn->last_stable == 1 && btn->stable_state == 0);
}

// 检测释放事件（低→高）
bool button_is_released(ButtonDebounce *btn) {
    return (btn->last_stable == 0 && btn->stable_state == 1);
}

// 使用示例
// ButtonDebounce btn1 = {0};
// void SysTick_Handler(void) {
//     bool raw = HAL_GPIO_ReadPin(BTN1_GPIO_Port, BTN1_Pin);
//     button_debounce(&btn1, raw);
// }
// void main_loop(void) {
//     if (button_is_pressed(&btn1)) { LED_Toggle(); }
// }
```

**详细解析：**

设计要点：
1. **非阻塞**：不使用 delay_ms，而是依赖定时器中断 1ms 计数
2. **边沿检测**：通过 `last_stable` 和 `stable_state` 比较检测按下/释放
3. **逻辑**：只在电平稳定持续 20ms 后才确认状态变化

如果需要节省资源，也可以使用 3 次采样投票法（3 次采样中 ≥2 次为 1 则确认为 1），延时仅约 2-3ms。

## 第 3 题答案 — 选择题 — ★☆☆

**正确答案：B**

**详细解析：**

边沿对齐 PWM 的基本公式：
- **PWM 频率** = TIM_CLK / [(PSC + 1) × (ARR + 1)]
- **占空比** = CCR / (ARR + 1) × 100%
- **分辨率** = log₂(ARR + 1) 位

示例：TIM_CLK = 72MHz，PSC = 71，ARR = 999
- 计数器频率 = 72MHz / 72 = 1MHz
- PWM 频率 = 1MHz / 1000 = 1kHz
- 若 CCR = 500，占空比 = 500/1000 = 50%

中心对齐模式（Center-Aligned）下，频率减半但波形对称性更好（谐波更少），适合电机控制。

## 第 4 题答案 — 选择题 — ★☆☆

**正确答案：B**

**详细解析：**

`assert()` 用于捕获"不应该发生"的编程错误：
- 前置条件检查：`assert(ptr != NULL);`
- 不变量检查：`assert(count <= MAX_ITEMS);`
- 逻辑断言：`assert(state < NUM_STATES);`

发布版禁用的原因：
1. 代码体积：每个 assert 增加几十字节
2. 执行时间开销
3. 产品中触发 assert 不如优雅降级（看门狗复位或进入安全模式）

**嵌入式自定义 assert**：
```c
#ifdef DEBUG
#define MY_ASSERT(expr) \
    do { if (!(expr)) { \
        printf("ASSERT: %s:%d: %s\n", __FILE__, __LINE__, #expr); \
        while(1); /* 暂停以便调试器查看 */ \
    } } while(0)
#else
#define MY_ASSERT(expr) ((void)0)
#endif
```
