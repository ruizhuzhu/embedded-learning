A. Watchpoint 在指定内存地址被读/写时触发；普通断点在指定代码地址被执行时触发。硬件 Watchpoint 数量通常只有 2-6 个

4. 每条日志包含：时间戳（tick）、日志等级、消息字符串

typedef struct {

    LogLevel level;

void log_write_isr(DebugLog *log, LogLevel lv, const char *fmt, ...);
